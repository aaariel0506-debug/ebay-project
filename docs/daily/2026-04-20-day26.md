## T2 完成：OrderSyncService ✅

### 功能

`modules/finance/order_sync_service.py` — eBay Fulfillment API 订单同步服务：

**API 端点**：`GET /sell/fulfillment/v1/order`

**核心流程**：
```
sync_orders(date_from, date_to)
→ 分页拉取订单（continuationToken）
→ _upsert_order() 每条订单
    → 解析 lineItems（含 sku / quantity / lineItemCost）
    → 提取 eBay fee（从 itemTxSummaries 或 Finances API fallback）
    → upsert Order 表（insert 或 update）
    → _write_transactions() 写入 SALE + FEE + SHIPPING 流水
```

**Transaction 流水类型**：
| 类型 | 说明 | 金额 |
|------|------|------|
| SALE | 商品销售 | 正数 |
| FEE | eBay 平台费 | 负数 |
| SHIPPING | 运费 | 正数 |

**幂等性**：同一 `ebay_order_id` 的订单第二次同步执行 update，不报错，不产生重复记录。
