# Day 7 — 2026-04-15

## 目标
Listing 模块骨架 + 单条 Listing 创建（3 步 API 流程）

## 完成内容

### modules/listing/ — 新建
- `schemas.py` — Pydantic 请求/响应模型
  - `ListingCreateRequest`：sku/title/condition/listing_price/quantity/description 等
  - `InventoryItemRequest` / `OfferRequest` / `ListingCreateResponse` 等
- `utils.py` — 常量 + 辅助函数
  - `EBAY_CONDITION_MAP` / `EBAY_CONDITIONS` / `EBAY_MARKETPLACE_IDS`
  - `normalize_condition()` — 支持 "new" → "NEW" 友好转换 + eBay 标准值验证
  - `build_inventory_availability()` / `build_offers_pricing_summary()`
  - `format_price()` / `validate_image_urls()` / `extract_listing_id_from_href()`
- `service.py` — `ListingService`
  - `create_single_listing()`：3 步 API 流程 + 自动回滚 + DB 写入 + 事件发布
  - Step 1: `PUT /inventory_item/{sku}` → eBay Inventory API
  - Step 2: `POST /offer` → eBay Offer API
  - Step 3: `POST /offer/{offer_id}/publish` → eBay Publish API
  - 失败时自动回滚（删除已创建的 inventory item）
  - 成功时写入 `EbayListing` + `Product` 表
  - 发布 `LISTING_CREATED` 事件到 EventBus
- `__init__.py` — 模块导出

### tests/test_listing_service.py — 新建（23 tests）
- `TestListingSchemas`（6 tests）：Schema 验证、默认值、边界错误
- `TestListingUtils`（10 tests）：条件转换/Availability/Pricing/URL 提取/验证
- `TestListingServiceUnit`（6 tests）：Step 1/2/3 单元测试、错误处理
- `TestListingServiceIntegration`（1 test）：完整 3 步流程 + DB + 事件集成测试
