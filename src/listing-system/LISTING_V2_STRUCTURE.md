| 参数 | 值 | 单位 | 说明 |
|------|-----|------|------|
| 关税 | 10% | - | 商品金额的 10% |
| 关税手续费 | 2.1% | 关税基础上 | 关税额的 2.1% |
| 通关手续费 | ¥225 | 日元/笔 | 固定费用 |

### 利润计算公式

```yaml
# 成本构成（日元）
cost_jpy:
  purchase: J列（采购价格）
  shipping: U列（SP运费）
  customs: V列（关税 + 手续费）
  # 关税 = 采购价格 × 10%
  # 关税手续费 = 关税 × 2.1%
  # 通关手续费 = ¥225

# 收入构成（美元）
revenue_usd:
  sale_price: P列
  shipping_charged: R列
  total: T列 = P + R

# eBay 费用
ebay_fees:
  commission: T × 12.7%      # eBay 佣金
  overseas_fee: T × 1.35%    # 海外交易手续费
  ad_fee: T × H列            # 广告费（默认 5%）
  per_transaction: $0.40      # 交易固定费

# 利润（US外）
profit_non_us = 总收入(JPY) - 采购成本 - SP运费 - 关税 - eBay费用(US外)

# 利润（US）
profit_us = 总收入(JPY) - 采购成本 - SP运费 - 关税 - eBay费用(US)

# ROI 目标
roi_target:
  non_us: 20-35%    # Y列
  us: > Y列          # AB列 > Y列
  low_price: ≤60%   # 低单价（<¥1000）可更高
```

## 🔵 SKU 编码规则（确认）

```yaml
format: "01-YYMM-NNNN"
store_id: "01"          # 固定
year_month: "YYMM"      # 如 2601 = 2026年1月
sequence: "NNNN"         # 0001-9999 递增

# 变体后缀
variant_suffix:
  color: "_Wh", "_Da", "_Pi", "_ye"
  color_code: "_#1", "_#12"
  size: "_A5", "_A6", "_We"
  other: "-SS"

# 当��最新编号
latest: "01-2603-XXXX"  # 2026年3月
```

## ⚫ 定价规则（确认）

```yaml
pricing:
  # 售价设置（P列）
  sale_price:
    target_roi_non_us:
      min: 20%
      max: 35%
      low_price_exception: 60%  # 采购价 < ¥1000
    
    constraint:
      roi_us > roi_non_us       # AB列 > Y列
  
  # 运费定价（R列）
  shipping_price:
    base: S列（增加成本 US）
    markup: "向上浮动 1-2 档"
    
    # 常用运费档位（美元）
```
