# Day 17 — 2026-04-15

## 目标
自动补货建议 + eBay 库存调整接口（Phase 3 Day 5）

## 完成内容

### `modules/inventory_online/restock_advisor.py` — 新建
- `RestockItem` dataclass：单个 SKU 的补货建议
- `RestockAdvisor.get_restock_list()`：
  - 分析近 N 天销售数据，计算日均销量
  - 预计售罄天数 = 当前库存 / 日均销量
  - 分类：urgent（< 7 天）/ soon（< 14 天）/ normal / unknown
  - 建议补货量 = 日均销量 × 目标天数 - 当前库存
  - 预估补货成本
- `RestockAdvisor.print_report()`：格式化打印补货建议报告

### `modules/inventory_online/quantity_adjuster.py` — 新建
- `AdjustmentResult` / `BatchAdjustmentResult` dataclass
- `QuantityAdjuster.adjust_ebay_quantity(sku, new_quantity)`：
  - 调用 eBay Inventory API 更新库存
  - 更新本地 EbayListing 表
  - 发布 LISTING_UPDATED 事件
  - 审计日志
- `QuantityAdjuster.batch_adjust_from_csv(csv_path, dry_run)`：CSV 批量调整
- 防重复扣减（用 related_order 去重）
