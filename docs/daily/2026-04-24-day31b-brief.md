## 目标

基于 Day 28.5 确认的真实 API 结构，实现：
1. **AD_FEE 采集**：从 Finances API `NON_SALE_CHARGE` transaction 提取
2. **SALE_TAX 采集**：从 Finances API SALE transaction 的 `ebayCollectedTaxAmount` 提取
3. **buyer_paid_total 采集**：调用已定义的 `_extract_buyer_paid_total()` 方法

## 实现方案

### 1. 新增 `_extract_ad_fee_from_finances` 方法

```python
def _extract_ad_fee_from_finances(self, order_id: str) -> Decimal:
    """从 Finances API 提取 AD_FEE"""
    try:
        resp = self._client.get(
            "/sell/finances/v1/transaction",
            params={"orderId": order_id},
        )
        for txn in resp.get("transactions", []):
            if txn.get("transactionType") == "NON_SALE_CHARGE" and txn.get("feeType") == "AD_FEE":
                return _decimal(txn.get("amount", {}).get("value", 0))
    except Exception:
        pass
    return Decimal("0")
```

### 2. 新增 `_extract_sale_tax_from_finances` 方法

```python
def _extract_sale_tax_from_finances(self, order_id: str) -> Decimal:
    """从 Finances API 提取 SALE_TAX（ebayCollectedTaxAmount）"""
    total_tax = Decimal("0")
    try:
        resp = self._client.get(
            "/sell/finances/v1/transaction",
            params={"orderId": order_id},
        )
        for txn in resp.get("transactions", []):
            if txn.get("transactionType") == "SALE":
                tax = _decimal(txn.get("ebayCollectedTaxAmount", {}).get("value", 0))
                total_tax += tax
    except Exception:
        pass
    return total_tax
```

### 3. 修改 `_upsert_order`

- 调用 `_extract_ad_fee_from_finances(order_id)` 替换 `total_ad_fee = Decimal("0")`
- 调用 `_extract_sale_tax_from_finances(order_id)` 替换 `total_sale_tax = Decimal("0")`
- 调用 `_extract_buyer_paid_total(data)` 替换直接读 `pricingSummary.total`
